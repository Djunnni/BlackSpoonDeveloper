## 실제 API로 전환하기

Mock 모드를 해제하고 실제 API를 사용하려면:

### 1. `.env` 파일 수정
```env
# Mock 모드 비활성화
VITE_USE_MOCK_API=false

# 실제 API URL 설정
VITE_API_BASE_URL=https://your-api-domain.com
```

### 2. 앱 재시작
변경 후 개발 서버를 재시작하면 실제 API를 사용합니다.

### 3. 로그인 페이지
실제 API 모드에서는 로그인 페이지(`/login`)가 활성화되며, 인증이 필요합니다.

### 문제 해결

**Q: 페이지가 로딩만 되고 데이터가 안 보여요**
A: 브라우저 콘솔을 열어 에러 메시지를 확인하세요. `.env` 파일에서 `VITE_USE_MOCK_API=true`가 설정되어 있는지 확인하세요.

**Q: 지역을 선택했는데 저장이 안 돼요**
A: Mock 모드에서는 로컬 상태로만 저장됩니다. 페이지를 새로고침하면 초기화됩니다.

**Q: 존을 선택했는데 반영이 안 돼요**
A: Mock 모드에서는 `nextZone`(내일 투자할 존)만 변경됩니다. `currentZone`(오늘 투자중인 존)은 00:00에 자동으로 전환되는 설정입니다.
